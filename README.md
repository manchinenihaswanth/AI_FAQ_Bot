# AI Food FAQ Chatbot

An AI-powered FAQ chatbot built using Python, Streamlit, NLP, and Cosine Similarity Matching.

## Features

* Food-related FAQ chatbot
* Smart answer matching
* NLP-based similarity checking
* User-friendly Streamlit interface

## Technologies Used

* Python
* Streamlit
* NLTK
* Scikit-learn

## How to Run

Install required libraries:

```bash
pip install streamlit nltk scikit-learn
```

Run the project:

```bash
streamlit run app.py
```
